# Zophiel - OSINT Link Toolkit

This repository contains the Zophiel OSINT Link Toolkit, a collection of tools for open-source intelligence gathering.

## Zohar Toolkit

The core of this repository is the `Zohar_Toolkit`, a powerful set of scripts and modules for various OSINT tasks.

### Features

*   **Identity Reconnaissance:** Tools for checking breaches, resolving entities, and creating dossiers.
*   **Content Generation:** Automated content creation modules.
*   **API Fuzzing and Analysis:** Scripts for discovering and analyzing APIs.
*   **And much more...**

Feel free to explore the `Zohar_Toolkit` directory to discover all the available tools.
